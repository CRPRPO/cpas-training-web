# CPAS虛擬個案晤談蒐證模擬器 V10

## V10更新重點

1. 上方功能按鈕改為四個，順序為：
   - 查看個案背景
   - 填寫報告內容
   - CPAS重點
   - 查看適性工作職務建議

2. 「本個案CPAS重點」改成獨立按鈕彈窗，只顯示分數摘要，不顯示判讀內容。

3. 修正「具創造性工作3分」解釋：
   - 不再只從創意美感解釋。
   - 改為跳脫框架、改變既有做法、提出新解法、重新組合資源等工作樣態。

4. 修正「營業工作0分」解釋：
   - 不再只從業務銷售職務解釋。
   - 改為需要主動說服他人、爭取支持、推動對方接受方案、承擔成果壓力與目標導向任務達成的工作樣態。

## Google Sheet / Apps Script

V10沿用V9的Google Sheet分欄送出格式：

送出時間｜小組姓名｜個案背景｜現況說明｜職涯問題｜CPAS人格特質重點｜五大適性工作判斷｜領導潛能｜顧問建議｜晤談素材｜完整報告

若仍只收到ABCD四欄，請確認 Apps Script 是否已重新部署新版本，而不是只按儲存。
