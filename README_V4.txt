# CPAS虛擬個案晤談蒐證模擬器 V4

## V4更新
1. 依據最新「虛擬個案側寫」更新第一頁個案背景：
   - 性別／年齡
   - 婚姻／居住
   - 學歷／科系與畢業時間
   - 知識與技能基礎
   - 畢業後摸索期
   - 主要正式工作經歷
   - 求職狀態、來站原因與主要求職期待

2. 題庫由24題增加為34題。
   新增第七區塊「求職動機與態度評估」，包含：
   - 求職動機
   - 求職意願程度
   - 可開始工作的時間
   - 求職行動意願
   - 求職態度與防衛
   - 可妥協條件
   - 不可妥協條件
   - 就業後最擔心的事
   - 工作成就感來源
   - 對工作的最低期待

3. 素材分類增加：
   - 求職動機
   - 求職意願
   - 求職態度
   - 可妥協條件
   - 工作成就感
   - 工作價值觀
   - 經濟壓力
   - 家庭因素
   - 行動準備度
   - 穩定就業風險

## 使用方式
解壓縮後，直接用 Chrome 或 Edge 開啟 `index.html`。
也可將所有檔案放到 GitHub Pages 使用。
