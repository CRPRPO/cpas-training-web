# CPAS虛擬個案晤談蒐證模擬器 V6

## V6更新
新增「查看適性工作職務建議」功能：
- 左側問題區新增按鈕。
- 右下角新增浮動按鈕。
- 以獨立彈出視窗呈現，不混入原本CPAS揭示頁。
- 內容包含：
  1. 定型工作2：分數意義與建議職務
  2. 對人工作1：分數意義與建議職務
  3. 營業工作0：分數意義與建議職務
  4. 非定型工作0：分數意義與建議職務
  5. 具創造性工作3：分數意義與建議職務
  6. 領導潛能0：分數意義與建議職務
  7. 整合五大適性工作的高層次建議
  8. 最建議的職涯切入定位
  9. 給就服員的媒合提醒

## 使用目的
讓學員在撰寫「五大適性工作」與「建議職務」時，可以隨時查閱職務方向，但不干擾盲蒐證流程。
